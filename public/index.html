Ui Kesäksi Kuntoon!

Tulokset ovat selvillä!

Kevään 2018 Ui kesäksi kuntoon -kampanja päättyi 31. maaliskuuta. 84 uimahallia ja 24 uimaseuraa ilmoitti tuloksensa uimaliittoon.

Yhteensä kampanjaan osallistui yli 16 tuhatta henkilöä, uintikilometrejä kertyi yli 55 tuhatta ja vesiliikuntatunteja yli 56 tuhatta!

Katso kaikki uimahallien tulokset ja uimaseurojen tulokset!

Mikä on Ui kesäksi kuntoon -kampanja?

Ui kesäksi kuntoon -kampanjassa kerätään kuukauden ajan metrejä uiden ja minuutteja vesijuosten tai vesijumpaten. Uimahallit kilpailevat leikkimielisesti keskenään eri kokoisten hallien sarjoissa (pienet, keskisuuret, suuret). Kolme aktiivisinta uimahallia palkitaan kussakin sarjassa.

Kampanjan tavoitteena on tukea uimahallien toimintaa ja lisätä uimahallien näkyvyyttä sekä aktivoida uusia kävijöitä halleihin. Kampanjan aikana uimahalleissa toteutetaan erilaisia tapahtumia, joihin eri kävijät ovat tervetulleita kokeilemaan erilaisia vesiliikuntamuotoja.

Kymmeniä tuhansia osallistujia!

Vuonna 2017 Ui kesäksi kuntoon -kampanjassa oli mukana 100 uimahallia, joissa kirjattiin noin 17 tuhatta kampanjaan osallistunutta liikkujaa!
